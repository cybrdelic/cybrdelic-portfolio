# Cybrdelic Portfolio

A minimalist, cyberpunk-themed portfolio website built with Rust and HTMX.

## Features

- Server-side rendering with Rust and Axum
- Dynamic content loading with HTMX
- Cyberpunk-inspired design
- Responsive layout
- Terminal-style animations

## Prerequisites

- Rust (latest stable)
- Cargo

## Setup

1. Clone this repository
2. Run `cargo build`
3. Run `cargo run`
4. Visit `http://localhost:3000`

## Development

The project structure is organized as follows:

```
cybrdelic-portfolio/
├── src/
│   └── main.rs
├── static/
│   ├── styles/
│   │   └── main.css
│   ├── js/
│   │   └── main.js
│   └── index.html
├── Cargo.toml
└── README.md
```

## Customization

- Update content in `index.html`
- Modify styles in `static/styles/main.css`
- Add new routes in `src/main.rs`
